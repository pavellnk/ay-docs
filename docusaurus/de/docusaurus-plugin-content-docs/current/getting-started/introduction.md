# Einführung in Ayunis Core

Willkommen zur offiziellen Dokumentation von **Ayunis Core**. Diese Anleitung hilft Ihnen, Ayunis Core zu verstehen, zu installieren und in Ihre Projekte zu integrieren.

## Was ist Ayunis Core?

Ayunis Core ist die grundlegende Bibliothek, die das Ayunis-Ökosystem antreibt. Sie bietet wesentliche Funktionen und Bausteine für die Erstellung robuster Anwendungen.

## Hauptfunktionen

- **Typsicher**: Mit TypeScript für eine ausgezeichnete Entwicklererfahrung gebaut
- **Modular**: Verwenden Sie nur das, was Sie brauchen
- **Erweiterbar**: Einfach anzupassen und zu erweitern
- **Gut dokumentiert**: Umfassende Dokumentation und Beispiele
